## Overview

This repository contains hands-on examples and implementations of advanced Java concepts, focusing on:
- Concurrency and multithreading
- Modern Java features (Records, Sealed Classes, Virtual Threads)
- Design patterns
- Thread synchronization mechanisms
- Enterprise Java patterns

## Topics Covered

### Concurrency & Locks
- **Locks**: ReentrantLock, ReadWriteLock, StampedLock
- **Synchronization**: Synchronized blocks, Conditions, Semaphores
- **Patterns**: Producer-Consumer, Connection Pooling
- **Issues**: Deadlock detection and prevention

### 🧵 Threads
- ThreadPoolExecutor
- ScheduledThreadPoolExecutor
- Work Stealing Pool
- Virtual Threads (Java 21+)
- Scoped Values

### 🎯 Design Patterns
- **Singleton Pattern**:
  - Lazy initialization
  - Eager initialization
  - Synchronized singleton
  - Double-checked locking (DCL)
  - Bill Pugh (Holder) pattern

### 🔧 Modern Java Features
- **Records**: Immutable data carriers
- **Sealed Classes**: Restricted class hierarchies
- **Enums**: Advanced enum usage and patterns

### 💾 Core Java Concepts
- Serialization
- Reflection
- Enum patterns
